Water Intake Tracker

CONCEPT: There is currently a health craze consuming the nation. However strange, there is an opportunity to design applications and products that help these users achieve their goals. While there is significant debate around what a "healthy" life/diet/routine actually looks like, something that almost all health professionals agree on is that water is VITAL. I created a water bottle attachment that prompts the user to drink, reinforces positive behavior with prompts, and tracks how many times they drink throughout the day.

TECHNICAL DETAILS: Unfortunately, I am not a genius and this is all just a ~prototype~. A user inputs values via Serial (input aspect), which then appear on the 4-Digit 7-Segment Display (output aspect). The user can input, through the Serial display, values from 0-9 by simply pressing those buttons on their keyboard. In order to display more complicated digits/words, the user must type a key-character. This is because the code only recognizes chars, not strings or integers.

The key is as follows:

q: 10

n: nice

h: hello:

b: bye

s: sip

For each Serial input char, there is an if/else statement, that if it tests true, turns on the corresponding digits and segments. The code is super long, so it is at the bottom of this page.
